Analyze this project and provide updated visual direction recommendations based on current 2026 design trends from Dribbble, Behance, Land-book, and Godly.

Provide specific recommendations for:

1. COLOR DIRECTION — Based on the product category and audience, recommend: a primary palette (2026 trending colors in this category), an accent color for CTAs and highlights, background treatment (pure white vs off-white vs dark mode default). Include specific HEX values and the emotional/brand rationale for each choice.

2. TYPOGRAPHY UPDATE — Recommend a current heading + body font pair. Consider: variable fonts for performance, personality match with the brand, Google Fonts availability. Provide the CSS @import and font-family declarations.

3. LAYOUT COMPOSITION — Identify the dominant layout style that fits this product: asymmetric editorial, bento grid, full-bleed hero, minimal centered, or multi-column magazine. Show a wireframe sketch using ASCII or describe the key layout decisions.

4. IMAGERY & ILLUSTRATION STYLE — What visual style fits best: photography (lifestyle/product/abstract), 3D illustration, flat vector, glassmorphism shapes, gradient blobs, or no imagery (pure typography)? Suggest 3 specific Unsplash search terms or illustration style keywords.

5. MOTION LANGUAGE — Recommend 2-3 micro-animations appropriate for the brand: entrance animations (fade-up, scale-in, slide), hover states, and transition durations/easing curves.

After recommendations, identify 3 specific pages/sections in this project that most need visual refresh and provide a prioritized action list.
