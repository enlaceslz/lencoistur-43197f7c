Review this project against current award-winning design trends (Q1/Q2 2026) from Awwwards, CSSDA, Godly, and SiteInspire.

Evaluate the project against these 2026 trend categories:

1. MOTION & SCROLL STORYTELLING — Does the site use scroll-driven animations (CSS scroll-timeline or JS IntersectionObserver)? Are there entrance animations that feel intentional (not just fade-in everything)? Identify 3 sections that would benefit from scroll-triggered reveals.

2. EXPRESSIVE TYPOGRAPHY — Is typography being used as a design element (large display type, variable fonts, kinetic text)? Or is it purely functional? Suggest one headline treatment that uses typography more expressively.

3. DARK MODE DESIGN — If the app has dark mode, is it a true dark design (not just color inversion)? Dark mode should use: near-black backgrounds (not pure #000), subtle surface layering (--background, --surface, --surface-raised), colored shadows instead of black shadows.

4. BENTO GRID LAYOUTS — For feature sections or dashboards, would a bento grid (mixed-size cards on a grid, some spanning 2 columns) improve the visual interest? Identify one section to convert.

5. GLASSMORPHISM & FROSTED SURFACES — Are glass effects used with restraint (max 2-3 elements per screen, always with sufficient contrast on the content)? If overused, identify which elements to remove it from.

6. AI-NATIVE INTERFACES — If this is an AI product, does the UI reflect current AI product design conventions: streaming text, confidence indicators, citation markers, tool call visualization?

For each trend: rate the current implementation (0-10), explain what is missing, and provide specific CSS/component code to implement the improvement.
