name: Website Audit & Relaunch
description: Análise sistemática em 5 dimensões: técnica, usabilidade, conteúdo, SEO e conversão — com plano de relaunch.
---

Run a systematic website audit across 5 dimensions and create a relaunch plan.

DIMENSION 1 — TECHNICAL ANALYSIS
Check: page load speed (target under 3 seconds), mobile usability (no horizontal scroll, readable text without zooming), broken links (404s in the codebase), HTTPS (all resources served over HTTPS), and current CMS/framework version.

DIMENSION 2 — USABILITY ANALYSIS
Check: navigation paths (can users find the main goal in under 3 clicks?), all forms working on mobile, CTA visibility above the fold, search functionality (exists if more than 20 pages), Nielsen heuristics (identify top 3 violations).

DIMENSION 3 — CONTENT ANALYSIS
Check: content currency (are there references to outdated information?), structure (proper heading hierarchy, scannable paragraphs), duplicates (similar pages that could be consolidated), and content gaps (what user questions are not answered?).

DIMENSION 4 — SEO ANALYSIS
Check: meta titles (unique, under 60 chars, keyword-first), meta descriptions (unique, under 160 chars, includes CTA), internal linking (orphan pages?), crawlability (robots.txt, sitemap.xml), and any obviously missing structured data.

DIMENSION 5 — CONVERSION ANALYSIS
Identify the primary conversion goal. Check: is there a clear funnel? Where are users most likely to drop off? Is there social proof near conversion points? How many steps from landing to conversion?

After the audit, create a RELAUNCH PLAN:
- Prioritized fix list: Critical (must fix) / High (fix soon) / Medium (next sprint)
- SEO migration checklist: if URLs will change, list required 301 redirects
- Post-launch monitoring: which metrics to track in the first 30/60/90 days

IMPORTANT: Audit only — do NOT modify code without explicit confirmation.
